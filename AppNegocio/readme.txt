Para conectar clonar git en la carpeta elegida
Posicionarse dento de la carpeta correspondiente y correr el servidor (python manage.py runserver)
El navegador da acceso a Clientes, Proveedores y articulos.
Desde cada uno se puede:
    Agregar nuevo
    Realizar una busqueda
    Ver el listado total (con la posibilidad de editar y/o eliminar) 


Los campos de fecha en fromularios de clientes y proveedores deben cargarse con formato mm/dd/aaaa
